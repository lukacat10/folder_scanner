# Usage:
Python 3.11.2 recommended.

If you want a venv:  
```bash
python -m venv venv
```

Normal usage installation:  
```bash
pip install -e .
```

Development installation:  
```bash
pip install -e .[dev]
```

Usage:  
```bash
folder_scanner [-h] [--extensions EXTENSIONS [EXTENSIONS ...]] [--output OUTPUT] path
```
